# HelloWorldASM

The program "Hello world" written in assembler native to MacOSX64

### Getting Started ###
Compile command:
    
        as -arch x86_64 -o hello_as_64.o hello_as_64.asm
 
 Link Command:
    
        as -arch x86_64 -o hello_as_64.o hello_as_64.asm
 
 Execute Command: 
    
        as -arch x86_64 -o hello_as_64.o hello_as_64.asm
 
 This should work =)
